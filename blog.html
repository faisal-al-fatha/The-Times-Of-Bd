1. Difference between var, let and const

Var Let const
var declarations are globally scoped or function/locally scoped. Variables declared with the let keyword are block-scoped. Variables declared with const keyword are also block-scoped.
Variables declared with the var keyword are hoisted. Variables declared with the let keyword are not hoisted. Variables declared with the const keyword are not hoisted.
var variables can be re-declared and updated let can be updated but cannot re-declared. const cannot be updated or re-declared

3. Why do we use template strings in JavaScript?

Template strings are a powerful feature of modern JavaScript released in ES6. It lets us insert/interpolate variables and expressions into strings without needing to concatenate like in older versions of JavaScript. It allows us to create strings that are complex and contain dynamic elements.As we know it's easy to make syntax errors with the old concatenation syntax if we have complex variables and expressions. Therefore template strings are a great improvement from what we had before.

Another great feature of template strings is that we can have strings with multiple lines for better code readability.
Template strings can be nested in each other. This is a great feature because many people want to create dynamic strings. Template strings allow us to nest regular strings or template strings within template strings.
